# Transcosmos Vietnam — Landing Page

A responsive landing page for **Transcosmos Vietnam**, exported from Figma Make and built with React, Vite, and Tailwind CSS.

🔗 Original Figma Design: [Transcosmos Vietnam Landing Page Version 2](https://www.figma.com/design/Ci3YCb2GFWq7VBPtnrZJCg/Transcosmos-Vietnam-Landing-Page-Version-2)

---

## Tech Stack

- **React 18** — UI framework
- **Vite** — build tool and dev server
- **Tailwind CSS v4** — utility-first styling
- **shadcn/ui + Radix UI** — accessible component primitives
- **MUI (Material UI)** — additional UI components
- **Framer Motion** — animations

---

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher
- npm, yarn, or pnpm

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Starolyzz/Transcosmos-Viet-Nam.git
cd "Transcosmos-Viet-Nam/Transcosmos Vietnam Landing Page Version Final"

# 2. Install dependencies
npm install

# Also install React if not automatically resolved
npm install react react-dom
```

### Running Locally

```bash
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
npm run build
```

The production-ready files will be output to the `dist/` folder.

This project can be deployed to any static hosting platform:

| Platform | Steps |
|----------|-------|

## Notes for Developers

- This codebase was **generated by Figma Make** — component structure mirrors the original Figma design
- Styles use a combination of **Tailwind utility classes** and **CSS variables** defined in `theme.css`
- The `src/app/components/ui/` folder contains auto-generated shadcn/ui components — avoid editing these directly
- Images may reference external Figma CDN URLs — replace with locally hosted assets before production deployment
